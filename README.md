## Тестовое задание

### [bankcards](https://github.com/nikolay-kukin/bankcards)

Spring Boot backend тестового задания для управления банковскими картами.

Реализованы JWT-аутентификация, роли `ADMIN` и `USER`, управление пользователями и картами, блокировка карт, переводы между своими картами, маскирование номера карты, AES/GCM-шифрование, Liquibase, Swagger и тесты.

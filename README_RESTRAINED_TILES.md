## Профессиональный профиль

Разрабатываю backend-сервисы на Java и Spring Boot.  
Работаю с PostgreSQL, ClickHouse, MinIO, Kafka и Docker.  
Использую REST, OpenAPI, Swagger, gRPC и HTTP/2 для сервисного взаимодействия.  
Go рассматриваю как дополнительное backend-направление.
